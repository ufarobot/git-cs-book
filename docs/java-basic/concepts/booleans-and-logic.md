

# Логические значения и условия (boolean)

Во многих задачах нужно отвечать на вопросы «да» или «нет»:  
число чётное или нечётное, попадает ли оно в диапазон, достаточно ли баллов и т.п.

В Java для таких ответов есть специальный тип `boolean` с двумя значениями:

- `true` — истина (да);
- `false` — ложь (нет).

Чаще всего логические значения используются в условиях `if` и циклах `while`.

```java
int x = 7;
boolean isEven = (x % 2 == 0);

if (isEven) {
    System.out.println("Чётное");
} else {
    System.out.println("Нечётное");
}
```

---

## Краткая сводка

Тип логического значения:

```java
boolean имя = значение;
```

Примеры:

```java
boolean done = false;
boolean isStudent = true;
```

Основные элементы:

- логический тип: `boolean`;
- значения: `true`, `false`;
- операции сравнения: `==`, `!=`, `<`, `>`, `<=`, `>=`;
- логические операции: `&&` (И), `||` (ИЛИ), `!` (НЕ).

---

## Тип boolean

`boolean` хранит только два значения: `true` или `false`.

```java
boolean isOpen = true;      // открыто
boolean isPassed = false;   // не прошёл
```

Часто логическое значение получается в результате сравнения:

```java
int score = 75;

boolean passed = score >= 60;    // true, если набрал 60 или больше
boolean excellent = score >= 90; // true, если 90 или больше
```

---

## Операции сравнения

Операции сравнения возвращают результат типа `boolean`.

Для чисел:

```java
int a = 5;
int b = 7;

boolean r1 = (a == b);   // равно?
boolean r2 = (a != b);   // не равно?
boolean r3 = (a < b);    // меньше?
boolean r4 = (a >= 0);   // больше или равно 0?
```

Частые варианты:

- `==` — «равно»;
- `!=` — «не равно»;
- `<`, `>`, `<=`, `>=` — сравнение по величине.

Для строк (`String`) сравнение по содержимому делается через метод `equals`, а не через `==`:

```java
String s1 = "yes";
String s2 = "yes";

boolean same = s1.equals(s2); // true
```

## Логические операции: И, ИЛИ, НЕ

Из нескольких условий можно собирать более сложные.

**Логическое И — `&&`**

Истина, только если оба условия истинны.

```java
int x = 10;

boolean inRange = (x >= 1) && (x <= 100);
// true, если x от 1 до 100 включительно
```

**Логическое ИЛИ — `||`**

Истина, если истинно хотя бы одно условие.

```java
int grade = in.nextInt();

boolean good = (grade == 4) || (grade == 5);
// true, если оценка 4 или 5
```

**Логическое НЕ — `!`**

Переворачивает значение: из `true` делает `false`, и наоборот.

```java
boolean isOpen = true;
boolean isClosed = !isOpen;  // false
```

## Логические выражения в if и while

Условие в `if` и `while` должно быть логическим выражением (`boolean`).

**Пример с if**

```java
int x = in.nextInt();

if (x > 0) {
    System.out.println("Положительное");
} else if (x < 0) {
    System.out.println("Отрицательное");
} else {
    System.out.println("Ноль");
}
```

**Проверка диапазона**

```java
int x = in.nextInt();

if (x >= 1 && x <= 100) {
    System.out.println("В диапазоне 1..100");
} else {
    System.out.println("Вне диапазона");
}
```

**Цикл while с логическим условием**

```java
int x = in.nextInt();

while (x > 0) {
    System.out.println(x);
    x = x - 1;
}
```

Цикл выполняется, пока условие `x > 0` истинно.

## Типичные шаблоны

**Чётное / нечётное число**

```java
int x = in.nextInt();

boolean isEven = (x % 2 == 0);

if (isEven) {
    System.out.println("Чётное");
} else {
    System.out.println("Нечётное");
}
```

**Попадание в диапазон**

```java
int x = in.nextInt();

boolean ok = (x >= 10) && (x <= 20);

if (ok) {
    System.out.println("ДА");
} else {
    System.out.println("НЕТ");
}
```

**Одно из нескольких значений**

```java
int month = in.nextInt();

boolean isWinter = (month == 12) || (month == 1) || (month == 2);
```

---

## Что нужно запомнить

Для ответов «да/нет» используется тип `boolean` со значениями `true` и `false`.

Сравнения (`==`, `!=`, `<`, `>`, `<=`, `>=`) и логические операции (`&&`, `||`, `!`) возвращают `boolean` и часто используются в условиях `if` и циклов `while`.

Сложные условия удобно собирать из простых:  
диапазоны — через `&&`, варианты «или то, или это» — через `||`, отрицание — через `!`.
