    Документация по Java Math API

    Класс java.lang.Math предоставляет методы для выполнения базовых математических операций, таких как тригонометрические функции, логарифмы, возведение в степень и другие. Этот класс является частью стандартной библиотеки Java и не требует создания экземпляров, так как все методы являются static.

    Обзор

    • Пакет: java.lang
    • Класс: Math
    • Основные возможности:
      • Константы: E, PI
      • Методы для арифметических, тригонометрических, экспоненциальных и логарифмических операций
      • Утилитарные методы, такие как abs, min, max и round

    Константы

    Константа Описание Значение
    E Основание натурального логарифма ~2.718281828459045
    PI Отношение длины окружности к её диаметру ~3.141592653589793

    Основные методы

    Ниже приведены наиболее часто используемые методы класса Math.

    Арифметические методы

    Метод Описание Пример
    static double abs(double a) Возвращает абсолютное значение числа типа double Math.abs(-5.0) → 5.0
    static double max(double a, double b) Возвращает большее из двух чисел типа double Math.max(3.5, 7.2) → 7.2
    static double min(double a, double b) Возвращает меньшее из двух чисел типа double Math.min(3.5, 7.2) → 3.5
    static double sqrt(double a) Возвращает квадратный корень числа типа double Math.sqrt(16.0) → 4.0

    Тригонометрические методы

    Метод Описание Пример
    static double sin(double a) Возвращает синус угла (в радианах) Math.sin(Math.PI / 2) → 1.0
    static double cos(double a) Возвращает косинус угла (в радианах) Math.cos(0) → 1.0
    static double tan(double a) Возвращает тангенс угла (в радианах) Math.tan(Math.PI / 4) → ~1.0

    Экспоненциальные и логарифмические методы

    Метод Описание Пример
    static double exp(double a) Возвращает число Эйлера, возведённое в степень a Math.exp(1.0) → ~2.718
    static double log(double a) Возвращает натуральный логарифм числа a Math.log(Math.E) → 1.0
    static double pow(double a, double b) Возвращает a, возведённое в степень b Math.pow(2.0, 3.0) → 8.0

    Методы округления

    Метод Описание Пример
    static long round(double a) Округляет до ближайшего числа типа long Math.round(3.7) → 4
    static double floor(double a) Возвращает наибольшее число типа double, меньшее или равное a Math.floor(3.7) → 3.0
    static double ceil(double a) Возвращает наименьшее число типа double, большее или равное a Math.ceil(3.1) → 4.0

    Пример использования

    public class MathExample {
    public static void main(String[] args) {
        double x = -10.5;
        double y = 4.0;

        System.out.println("Абсолютное значение: " + Math.abs(x)); // 10.5
        System.out.println("Квадратный корень: " + Math.sqrt(y)); // 2.0
        System.out.println("Возведение в степень: " + Math.pow(y, 2)); // 16.0
        System.out.println("Синус PI/2: " + Math.sin(Math.PI / 2)); // 1.0
        System.out.println("Округление: " + Math.round(x)); // -10
    }
}
